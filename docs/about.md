# 郑勇胜

### 架构实践者与布道者

我是一名程序员，曾先后在京东与华为担任架构师。

目前从事企业数字化转型咨询工作，在实际工作中发现了模型和实现的脱节，本体论提供了一个全新的解决思路。


### 技术主张

我始终认为，软件系统的核心价值不应被淹没在破碎的代码实现或空洞的 PPT 幻象中。特别是在 AI 浪潮重塑行业的当下，我们面临的不再是算力的贫瘠，而是**逻辑主权的丧失**。

通过 **OntologyAction**，我希望将多年来在架构变革中沉淀的实践思考——即“本体优先（Ontology-First）”范式——进行系统性输出。我的目标不是进行纯粹的哲学思辨，而是通过总结可迁移的工程方法论，帮助企业构建出**逻辑自证、边界确定**的数字镜像世界。

### 持续关注

除了大型软件架构，我依然保持着对计算机科学多个领域的持续跟进与研究。对我而言，管理与研究工作是架构实践的延伸，而编写代码则是保持逻辑清澈的唯一途径。
